# V7 Market Intelligence Update

This update streamlines the dashboard and focuses the workflow on actionable investment decisions:

1. Market Timing: risk-on / neutral / risk-off and suggested equity/cash allocation.
2. Sector Rotation: overweight / neutral / underweight / exit by sector ETF.
3. Stock Ranking: buy / hold / exit candidates ranked by stock score.
4. Exit Watchlist: names requiring de-risking or exit review.
5. Governance: model metrics, feature power and alpha quality are folded into one tab.

Removed / consolidated low-value duplicate tabs:
- Separate V5.5 Macro Credit / Economic Regime / Cross Asset / Earnings / Asset Allocation tabs.
- Separate V6 Model Engine / Feature Power / Alpha Quality / Signal Distribution tabs.

Useful information from those tabs is now shown in Credit-Macro, AI Research, Model Governance, and the four Market Intelligence tabs.
